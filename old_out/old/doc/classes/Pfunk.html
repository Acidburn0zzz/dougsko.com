Class Pfunk
In: pfunk.rb
Parent: Object

This class does two things. When run from the command line, it creates a new template for you to write in. The template has eRuby code in it and uses methods from this class as well to update its content. The usage of these methods can be seen in the template below.

Methods

Public Class methods

new()

This is the only thing you really need to change when starting a new site.

<% funk = Pfunk.new %>

Public Instance methods

list_files() {|file, nice_name| ...}

Lists the files for the nav bar. The glob grabs everything ending in .rhtml except the index file.

<% funk.list_files do |full, nice| %> 

    <% puts "<a href=\#{full}>\#{nice}</a><br />" %>

<% end %>

print_head()

Returns everything you need between <head></head>. It used @title to set the page‘s title.

<% puts funk.print_head %>

print_template(file)

Creates a template for a new page. This is only used when run from the command line.

$ ruby pfunk.rb -c index

reddify(text)

Returns the Red Cloth formatted version of the string provided.

<% puts funk.reddify("h1. hello") %>
